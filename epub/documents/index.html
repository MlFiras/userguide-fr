Module de création de documents

Cette section regroupe la documentation relative au module de création de documents de MapMint.

Le module de gestion des documents permet la publication sur la page d’accueil de divers documents qui apparaitront dans une section spécifique de la page d’accueil (troisième lien).

La page du module est divisée en deux parties :

  • la partie de gauche appelée le Panneau des documents, il liste l’ensemble des documents créés et permet l’ajout et la suppression de documents
  • la partie de droite appelée le Panneau d’information, il permet de saisir les informations relatives à un document
  • Panneau des documents
    • Ajouter un nouveau document
    • Supprimer un document
  • Panneau d’information
    • Nom du document
    • Thème du document
    • Fichier ou URL
    • Description du document
    • Droits des groupes
