Module de gestion des territoires

Cette section regroupe la documentation relative au module de gestion des territoires de MapMint.

Le module de gestion des territoires permet de gérer une hierarchie de territoires qui sera par la suite utilisée dans le module de gestion des indicateurs permettant de joindre un territoire Ã  une source de données tierce (base de données, requéte ou autres fichiers XLS,CSV ...).

La page du module est divisée en deux parties :

  • la partie de gauche appelée le Panneau des territoires, il liste l’ensemble des territoires créés et permet l’ajout et la suppression de territoires
  • la partie de droite appelée le Panneau d’information, il permet de saisir les informations relatives Ã  un document
