Module document creation

This section contains the documentation MapMint document creation module.

The document management module allows the publication on the home page of various documents to appear in a specific section of the home page (third link).

The page of the module is divided into two parts:

  • the left side called the: doc: docslist it list all the records created and allows adding and deleting documents

  • the right part called the: doc: infopanel he used to enter the information for a document

  • Document panel
    • Add a new document
    • Deleting a document
  • Information panel
    • Document Name
    • Theme of the document
    • File or URL
    • Description of Document
    • Rights groups
