Dashboard¶

This section contains the documentation for the dashboard MapMint.

The dashboard MapMint consists of three specific sections:

  • Overview which is the visible content in page load,

  • Users that allows the management of users and user groups,

  • Parameter that allows the management of application settings.

Warning

It is important to note that only users in the group super admin are allowed to access the modules User Management and Control Panel
